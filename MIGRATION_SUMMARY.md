## Что нужно сделать пользователю

### Шаг 1: Установить PostgreSQL
```bash
# Windows: скачать с https://www.postgresql.org/download/windows/
# Linux: sudo apt install postgresql postgresql-contrib
# macOS: brew install postgresql@15
```

### Шаг 2: Создать БД и загрузить данные
```bash
psql -U postgres -c "CREATE DATABASE otk_db;"
psql -U postgres -d otk_db -f database/schema.sql
psql -U postgres -d otk_db -f database/data.sql
```

### Шаг 3: Установить Npgsql пакет
```bash
cd WebAPI_OTK
dotnet add package Npgsql.EntityFrameworkCore.PostgreSQL --version 8.0.0
```

### Шаг 4: Обновить Program.cs
Заменить в файле `WebAPI_OTK/Program.cs`:
```csharp
// Было:
options.UseSqlServer(...)

// Стало:
options.UseNpgsql(...)
```

Добавить using:
```csharp
using Npgsql.EntityFrameworkCore.PostgreSQL;
```

### Шаг 5: Настроить пароль
В файле `WebAPI_OTK/appsettings.json` заменить `your_password` на реальный пароль PostgreSQL.

### Шаг 6: Запустить приложение
```bash
cd WebAPI_OTK
dotnet run
```

Открыть: http://localhost:5000
