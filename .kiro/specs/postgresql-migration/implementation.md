## Что требуется от пользователя

### Обязательные шаги

1. **Установить PostgreSQL**
   - Windows: скачать с официального сайта
   - Linux: `sudo apt install postgresql`
   - macOS: `brew install postgresql@15`

2. **Создать базу данных**
   ```bash
   psql -U postgres -c "CREATE DATABASE otk_db;"
   ```

3. **Выполнить SQL скрипты**
   ```bash
   psql -U postgres -d otk_db -f database/schema.sql
   psql -U postgres -d otk_db -f database/data.sql
   ```

4. **Установить Npgsql пакет**
   ```bash
   cd WebAPI_OTK
   dotnet add package Npgsql.EntityFrameworkCore.PostgreSQL --version 8.0.0
   ```

5. **Обновить Program.cs**
   - Заменить `UseSqlServer` на `UseNpgsql`
   - Добавить `using Npgsql.EntityFrameworkCore.PostgreSQL;`
   - См. `database/PROGRAM_CS_CHANGES.md`

6. **Настроить пароль**
   - В `appsettings.json` заменить `your_password` на реальный

7. **Запустить приложение**
   ```bash
   dotnet run
   ```
