## Устранение проблем

### Ошибка: "password authentication failed"
- [ ] Проверил пароль в `appsettings.json`
- [ ] Попробовал подключиться через psql с тем же паролем
- [ ] Если нужно, сбросил пароль: `ALTER USER postgres PASSWORD 'новый_пароль';`

### Ошибка: "database does not exist"
- [ ] Проверил, что БД создана: `psql -U postgres -l`
- [ ] Если нет, создал: `psql -U postgres -c "CREATE DATABASE otk_db;"`

### Ошибка: "could not connect to server"
- [ ] Проверил, что PostgreSQL запущен
- [ ] Windows: Services → postgresql-x64-15 → Start
- [ ] Linux: `sudo systemctl start postgresql`
- [ ] Mac: `brew services start postgresql@15`

### Ошибка: "relation does not exist"
- [ ] Проверил, что выполнен schema.sql
- [ ] Проверил таблицы: `psql -U postgres -d otk_db -c "\dt"`
- [ ] Если таблиц нет, выполнил schema.sql заново

### Ошибка HTTP 500 в браузере
- [ ] Открыл консоль браузера (F12)
- [ ] Посмотрел детали ошибки
- [ ] Проверил логи приложения в терминале
- [ ] Убедился, что все справочники загружены в БД
