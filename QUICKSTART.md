## Остановка приложения:

Если нужно остановить приложение, выполните в терминале:
```bash
Ctrl+C
```

Или используйте команду Kiro для остановки фонового процесса.

## Перезапуск приложения:

Если приложение остановлено, запустите его командой:
```bash
cd WebAPI_OTK
dotnet run
```
